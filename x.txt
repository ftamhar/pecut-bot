Jumlah Kabupaten/ Kota (Total)
Jumlah Kabupaten/ Kota (yang sudah disurvey) -> hasil filter di atas
Jumlah Fasyankes (Total) -> facility type id
Jumlah Fasyankes (yang sudah disurvey) -> hasil filter diatas

Wilayah	: Provinsi, Kabupaten, Kecamatan, hingga Kelurahan -> di list data change request
